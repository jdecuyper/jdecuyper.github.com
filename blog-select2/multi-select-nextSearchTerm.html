Select 2: enable 'nextSearchTerm' for multiple select2

This demo is based on following select2 SHA: 7087c4b.

The 'nextSearchTerm' can be used for multi select2 boxes.

Click on the search box, type in 'Gr' and select the 'Green' item. Then click again on the search box and you will see your search term displayed again ('Gr'). The 'nextSearchTerm' function can be customized to cover more complex user cases.
